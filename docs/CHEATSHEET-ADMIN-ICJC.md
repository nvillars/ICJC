# CHEATSHEET – Operación rápida /admin (ICJC)
**Fecha:** 2025-08-31 — **Zona horaria:** America/Lima

## URLs
- Panel: `https://TU-DOMINIO/admin`
- Login: `https://TU-DOMINIO/admin/login`

## Roles (resumen)
- **admin**: todo.
- **editor**: publica.
- **author**: crea/edita borradores propios.
- **moderator**: modera.
- **viewer**: sólo lectura.

## Estados
`draft → review → published → archived`

## 5 acciones más comunes
1) **Publicar Sermón**
   - Contenido ▸ Sermones ▸ Nuevo → completa campos → “Enviar a revisión” → Publicar.
2) **Crear Evento**
   - Contenido ▸ Eventos ▸ Nuevo → fecha/hora + lugar/mapa → Publicar.
3) **Post/Devocional**
   - Contenido ▸ Posts ▸ Nuevo → título/extracto/contenido → Revisión → Publicar.
4) **En Vivo**
   - Si no detecta live, pega el **permalink** del video de Facebook.
5) **Sincronizar Facebook**
   - /admin/sync/facebook → “Sincronizar ahora” → aprobar candidatos (drafts) → publicar.

## Buenas prácticas (checklist)
- [ ] Títulos claros y extracto breve (SEO).  
- [ ] **Alt** en todas las imágenes.  
- [ ] Fechas en **America/Lima**.  
- [ ] Tags y serie/predicador/pasaje completos.  
- [ ] Revisa en **Bitácora** cambios sensibles.

## Emergencias
- **No se ve video**: usa permalink **público**.  
- **No aparece en el sitio**: estado = published + fecha correcta.  
- **Login falla**: reset de contraseña con un *admin*.

Guárdalo como `docs/CHEATSHEET-ADMIN-ICJC.md`.
